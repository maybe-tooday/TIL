## 버전 관리랑 상관없는 파일

### .gitignore

- git 저장소에 `.gitignore` 파일을 생성하고 해당 내용을 관리
  - 작성 예시
    - 특정 파일 : `a.txt` , `test/a.txt`
    - 특정 디렉토리 : `/my_secret`
    - 특정 확장자 : `*.exe`
    - 예외 처리 : `!b.exe`
